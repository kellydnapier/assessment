# Case Management Assessment Generator

A browser-based clinical assessment generator for case managers. Select diagnoses, medications, provider details, and clinical data — then generate a complete narrative assessment with one click.

## Body Systems

- **Cardiac** — HTN, CHF, CAD, AFib, PAD, Cardiomyopathy, Valvular Heart Disease, Pacemaker/ICD, MI History, Stroke/TIA, DVT/PE
- **Pulmonary** — COPD, Asthma, OSA, Valley Fever, Pulmonary Hypertension, Pulmonary Fibrosis, Bronchiectasis, ILD, Pneumonia Hx, Pleural Effusion, TB Hx
- **Endocrine** — Type 1 & 2 Diabetes, Hypothyroidism, Hyperthyroidism, Hyperparathyroidism, Adrenal Insufficiency, Cushing's, Osteoporosis, Metabolic Syndrome
- **Renal** — CKD, ESRD, BPH, Kidney Stones, PKD, Glomerulonephritis, Nephrotic Syndrome, Recurrent UTIs, Prostatectomy Hx, Renal Artery Stenosis
- **Neurological** — CVA, TIA, Epilepsy, Parkinson's, MS, Alzheimer's/Dementia, Cocci Meningitis, Neuropathy, Migraines, VP Shunt
- **Gastrointestinal** — GERD, Crohn's, UC, Diverticulitis, IBS, Cirrhosis, Hepatitis, Pancreatitis, Celiac, GI Bleed, PUD
- **Musculoskeletal** — OA, RA, DJD, DDD, Gout, Fibromyalgia, Spinal Stenosis, Herniated Disc, Joint Replacement, Fracture Hx, Osteoporosis
- **Hematologic** — Iron Deficiency Anemia, B12/Folate Deficiency, Anemia of Chronic Disease, Sickle Cell, Thrombocytopenia, DVT/PE, Clotting Disorders, Polycythemia Vera, MDS, Hemophilia
- **Integumentary** — Basal Cell Carcinoma, Squamous Cell Carcinoma, Melanoma, Psoriasis, Eczema, Rosacea, Actinic Keratosis, Shingles
- **Behavioral Health** — MDD, GAD, Bipolar, Schizophrenia/Schizoaffective, PTSD, SUD, OCD, ADHD, Panic Disorder

## Usage

Open `index.html` in any modern browser. No server or build step required. Both `index.html` and `data.js` must be in the same directory.

## Deployment

This project is ready for GitHub Pages. Enable Pages in your repo settings (Settings → Pages → Deploy from main branch) and it will be live at `https://yourusername.github.io/repo-name/`.
